# NeuralRNN
NeuralRNN is a tool to learn and implement RNN in neuroscience for dynamics modeling and geometry analysis. It provides a unifying framework for the most advanced methods to optimize RNNs on a sensory/cognitive task or reconstruct the observed neural/behavioral activity.
